Containers

Containers are the most basic layout element in Bootstrap and are required when using our default grid system.

This div has a bs class of container. the container class gives a max-width at each responsive break point, and allows padding
This div has bs class of container-fluid. and has width of 100% at all breakpoitns
This is container-{breakpoint}. For this example I used lg for the large break point. What this means is this div will be 100% width until the browser hits the large breakpoint of 992px.
