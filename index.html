Media Query w/ min-width

"Mobile first" means you code your website for msobile devices first. When coding for mobile first sites, your default class will not need any media queries. That's because it is by default styled to fit small devices.

As you move up in screen sizes you'll then need to add media queries so your content better fits those screens.

Flex Container and Flex Items

  • The Flex Container has properties that instruct how the items will display. The instructions are properties of the parent.
  • Flex Items are properties of the children. Think of the items as the children.
